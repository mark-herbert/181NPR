Time In and Time Out of Residents

Quick Start ›› Lobby ››


1. By default, the Resident Time In/Out interface will automatically appear.

2. For the resident to time in/out, place the registered fingerprint of the resident in the fingerprint scanner.

2.1 If the fingerprint is recognized, the resident's name and status (logged in/out), will appear.

2.2 If the fingerprint is not recognized within three consecutive tries, the resident will type his/her ID number, then click .

3.2.1. If the the ID number is recognized, a Successful dialog box will appear, click .

3.2.2 If ID number is not recognized, Unsuccessful dialog box will appear, click to re-enter the resident ID number.

4. Click the to go back to the main menu of the Lobby Interface.
